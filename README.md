# majodev.com

majodev.com website source code hosted on majodev.github.io.

## Prerequites
These have to be installed globally:
- node / npm
- bower
- grunt

```bash
  npm install
  bower install
```

## Build (dev environment)
```bash
  grunt
```

go to localhost:8080.

## Build (productive)
```bash
  grunt productive
```
