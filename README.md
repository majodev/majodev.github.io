# ranf.tl

website source code hosted on majodev.github.io and **ranf.tl**!

## Prerequites
These have to be installed globally:
- node / npm
- bower
- grunt

```bash
  npm install
  bower install
```

## Build (dev environment)
```bash
  grunt
```

go to localhost:8080 and enjoy.

## Build (productive)
```bash
  grunt productive
```
