# Portfolio ### Some selected projects
###### Master thesis ### Physiogame: Leap Motion for exercise therapy #### Developing modular JavaScript applications May 2013 - January 2014 FH JOANNEUM information management master
*"Within this thesis the application Physiogame is developed. This application utilizes Leap Motion to address requirements in the fields of physiotherapy and occupational therapy. Moreover, Physiogame is implemented in JavaScript and this thesis covers the pitfalls, code conventions and common design patterns of this programming language. Likewise, concepts to achieve modularity, automation, a testing environment and dependency management have to be analyzed."* [PDF (german)](content/masterthesis/Diplomarbeit_Mario_Ranftl.pdf)
###### Project ### 20+ interactive exhibits and maintenance software #### for Universalmuseum Joanneum August 2012 - May 2013 FH JOANNEUM information management master
* Developing 19 exhibits & 1 presentation app * Easy CMS, external interfaces plugin * API documentation * Hands-On seminars
###### Project ### wisp: wipe information space #### Gesture controlled information system June 2012 FH JOANNEUM information management master
*"WISP provides a unique and intuitive way for the presentation of information on potentially small places and a playful approach to control external systems. The precise gesture control is the underlying basic concept."*
###### Project ### WLAN Security Revisited #### Rogue AP, Deauthentication Attack and Portal via PwnSTAR in 2 Minutes June 2012 FH JOANNEUM information management master
*"Creating a Rogue Access Point with redirecting web portal and hijacking open wifi users via de-authentication attack."*
###### Paper ### Mobile Devices Usability #### Good, Bad & Ugly / Heuristika August 2011 FH JOANNEUM information management bachelor
*"The aim of this dissertation is to create an usability-checklist, which is focused on the characteristics of mobile applications for touch devices. The outcome is a comprehensible compilation of heuristics formatted as a checklist."* [PDF (german)](content/akim/AKIM_Mario_Ranftl.pdf)
###### Bachelor thesis 2 ### Lower-CASE #### Evaluation of Database CASE-Tools for teaching at a graduate level May 2011 FH JOANNEUM information management bachelor
*"The aim of this degree dissertation is to gather a set of criteria for evaluating the optimal database Lower-CASE tool, which can be used for teaching databases at the FH JOANNEUM. The term Lower-CASE refers to tools that support later phases of a software development process, such as implementation and testing."* [PDF (german)](content/bac2/BAC-2_Mario_Ranftl.pdf) · [source](http://www.fh-joanneum.at/aw/home/Studienangebot_Uebersicht/department_angewandte_informatik/ima/Menschen/AbsolventInnen/~bbqe/Abschlussarbeiten_Details_ima/?arbid=4308062788&arbtyp=Bakk.-Arbeit&lan=en)
###### Bachelor thesis 1 ### iPhone Development Tools #### Analysis of development tools for the Apple iPhone September 2010 FH JOANNEUM information management bachelor
*"The aim of this work is to determine the advantages and disadvantages of selected IDEs, that are able to develop applications for the Apple iPhone. Alternative development-tools should be found and a structured overview should be prepared."* [PDF (german)](content/bac1/BAC-1_Mario_Ranftl.pdf) · [source](http://www.fh-joanneum.at/aw/home/Studienangebot_Uebersicht/department_angewandte_informatik/ima/Menschen/AbsolventInnen/~bbqe/Abschlussarbeiten_Details_ima/?arbid=4304674769&arbtyp=Bakk.-Arbeit&lan=en)
